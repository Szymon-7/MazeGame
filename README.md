# MazeGame - JavaFX Maze Runner

A maze runner game written in Java using JavaFX.  
This project uses ***Maven*** to manage dependencies and build the project.

---

## Requirements
- **Java 23 or later (Java 25 recommended)**  
Verify with:
`java -version`

- **Maven**  
Verify with: `mvn -v`

## Running the Game

From the project root run with Maven:  
`mvn javafx:run`

This automatically:  
**1.** Downloads ***JavaFX***  
**2.** Compiles all Java files  
**3.** Launches the game  

## Build Commands
Compile only:  
`mvn compile`

Clean build files:  
`mvn clean`
